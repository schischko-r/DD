# Mascot assets

Transparent PNG states for Data Driven interfaces:

- `happy.png` — success, welcome, completion;
- `love.png` — appreciation and positive feedback;
- `thinking.png` — analysis and loading;
- `question.png` — empty state or missing information;
- `exhausted.png` — complex or long-running work;
- `something-broke.png` — errors and failed states.

Import individual files or use the named exports from `index.js`.
